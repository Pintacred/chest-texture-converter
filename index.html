Chest Texture Converter  v0.3

This program converts the UV mapping of the single and double chest from Java Edition to Bedrock Edition and vice versa by rearranging the faces to their different corresponding positions and orientations.

Instructions are found below.

INSTRUCTIONS:

This program only accepts .png files.

If you're converting a single chest, the texture must be at least 64x64 and is a square. 64x65 will not work and you will get a broken image.

If you're converting a double chest from Java to Bedrock, both texture files must be at least 64x64 and are both squares, just like before. Both left and right textures should also be in the same resolution. For example, if the left side is 64x64 and the right side is 128x128, the texture will not convert properly.

If you're converting a double chest from Bedrock to Java, the texture must be at least 128x64 and is a horizontal rectangle. Again, if the dimensions of the image are off, the converted texture will be broken.

Resolution is the level of resolution of the texture. The default is 16x. You may notice that the chest texture file is 64x64 pixels, or for the double chest in bedrock, 128x64. In the program though, resolution is the amount of pixels in a full block. So if you're making a 32x texture pack, use resolution=32x.
